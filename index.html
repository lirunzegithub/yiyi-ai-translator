隐私政策

译亿AI（以下简称“本应用”）是一款 iOS 端的情景式 / 对话式翻译工具。我们高度重视你的隐私。 本应用没有任何自有服务器，开发者不收集、不存储、不上传你的任何个人数据。

1. 我们如何处理你的数据

  • 文字 / 图片翻译：默认由设备本地的离线模型与词典处理。若你在「设置」里自行配置了第三方 API（OpenAI 兼容接口），相关文字/图片会按你的配置发送到你自己填写的那个 API 地址，用于生成译文。该数据由你选择的 API 提供方按其隐私政策处理，与本应用开发者无关。
  • 语音识别：录音在设备本地完成转写（端侧模型），用于把语音转成文字再走上面的翻译流程。
  • 位置信息：仅在你主动使用相关功能时获取。用于：①为你的发送记录附带位置；②（可选）天气显示；③（可选）“安全模式”下把你的实时位置分享给你指定的接收人。位置默认仅保存在你的设备本地。
  • 会话历史：你的对话、录音、图片仅保存在设备本地（iOS SwiftData / 沙盒），可随时删除。
  • API 密钥：你填写的第三方 API 密钥仅加密保存在设备的钥匙串（Keychain），不外传。

2. 第三方服务

为实现功能，本应用可能与以下服务通信（仅传输实现该功能所必需的数据）：

  • 你自行配置的翻译 API 提供方（文字/图片译文，可选）。
  • Open-Meteo（天气，可选，会用到大致位置/经纬度）。
  • Hugging Face（按需下载可选的本地模型文件）。

这些第三方各有其隐私政策，请你自行查阅。

3. 我们不做的事

  • 不使用任何广告 SDK、第三方分析/统计 SDK。
  • 不使用 IDFA、不进行跨 App 跟踪。
  • 不为开发者收集或回传任何数据（本应用无后端）。

4. 你的控制权

  • 可在 iOS「设置 → 译亿AI」中随时撤销麦克风、定位、相册、语音识别等权限。
  • 可在应用内删除任意会话/录音/图片。
  • 无需注册账号，卸载应用即清除本地数据。

5. 儿童

本应用非面向 13 岁以下儿童设计，不会有意收集儿童个人信息。

6. 变更

隐私政策如有更新，将在本页面更新生效日期。

Privacy Policy

译亿AI (“the App”) is an iOS situational / conversational translation tool. We take your privacy seriously. The App has no servers of its own; the developer does not collect, store, or upload any of your personal data.

1. How we handle your data

  • Text / image translation: handled by on-device offline models and dictionaries by default. If you configure a third-party API (OpenAI-compatible) in Settings, the relevant text/images are sent to the API endpoint you yourself entered, solely to produce translations. That data is handled by the API provider you chose under their own privacy policy; it does not reach the App’s developer.
  • Speech recognition: transcription runs on-device; the resulting text then follows the translation flow above.
  • Location: accessed only when you actively use the related features, to: (a) tag your sent records with a location; (b) optionally show weather; (c) optionally, in “Safety Mode,” share your live location with recipients you designate. Location is stored only on your device by default.
  • Conversation history: your chats, recordings, and images are stored only on your device (iOS SwiftData / sandbox) and can be deleted anytime.
  • API keys: any third-party API key you enter is stored encrypted in the device Keychain only and never transmitted elsewhere.

2. Third-party services

To provide features, the App may communicate with (transmitting only what that feature requires):

  • Your configured translation API provider (text/image translations, optional).
  • Open-Meteo (weather, optional; uses approximate location/coordinates).
  • Hugging Face (on-demand download of optional on-device model files).

Each third party has its own privacy policy; please review them.

3. What we do NOT do

  • No advertising SDKs, no third-party analytics SDKs.
  • No IDFA, no cross-app tracking.
  • No data collected or sent back to the developer (the App has no backend).

4. Your controls

  • Revoke microphone, location, photos, and speech-recognition permissions anytime in iOS Settings.
  • Delete any conversation/recording/image inside the App.
  • No account required; uninstalling removes local data.

5. Children

The App is not directed to children under 13 and does not knowingly collect their personal information.

6. Changes

If this policy changes, the effective date on this page will be updated.
